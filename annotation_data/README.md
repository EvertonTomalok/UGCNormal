# Annotated dataset

## Tags:
- `O` --> common misspellings
- `Z` --> real-word misspellings
- `I` --> internet slang
- `C` --> case use (proper names and acronyms)
- `CC` --> case use (start of sentence)
- `S` --> glued words
- `P` --> punctuation

## Tagged text example:
- <wrong_word>`*<TAG>`[expected_word]
- videos`*O`[vídeos]
- da`*Z`[dá]
- ñ`*I`[não]
- skype`*C`[Skype]

ps: punctuation tag `*P[.]` just indicates missing periods.
